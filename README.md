# BotC Bremen & Umzu

Based on [Start Bootstrap - One Page Wonder](https://startbootstrap.com/theme/one-page-wonder/).
It is a basic one page template for [Bootstrap](https://getbootstrap.com/) created by [Start Bootstrap](https://startbootstrap.com/).

## Deploy

This repository is deployed via github pages on [botc-bremen.de](https://botc-bremen.de).

## Editing

Clone the source files and navigate into the directory. Run `npm install` and then run `npm start` which will open up a preview in your default browser, watch for changes to core template files, and live reload the browser when changes are saved. You can view the `package.json` file to see which scripts are included.

The single page source files is [src/pug/index.pug](src/pug/index.pug)

### npm Scripts

* `npm run build` builds the project - this builds assets, HTML, JS, and CSS into `dist`
* `npm run build:assets` copies the files in the `src/assets/` directory into `dist`
* `npm run build:events` compiles the events from the external source into `dist`
* `npm run build:pug` compiles the Pug located in the `src/pug/` directory into `dist`
* `npm run build:scripts` brings the `src/js/scripts.js` file into `dist`
* `npm run build:scss` compiles the SCSS files located in the `src/scss/` directory into `dist`
* `npm run clean` deletes the `dist` directory to prepare for rebuilding the project
* `npm run start:debug` runs the project in debug mode
* `npm start` or `npm run start` runs the project, launches a live preview in your default browser, and watches for changes made to files in `src`

You must have npm installed in order to use this build environment.
